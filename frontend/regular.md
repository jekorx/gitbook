# 正则表达式

#### 常用正则表达式

```js
// 手机号校验
/^1[3456789]\d{9}$/

// 邮箱校验
/^\w+((-\w+)|(\.\w+))*@[A-Za-z0-9]+((\.|-)[A-Za-z0-9]+)*\.[A-Za-z0-9]+$/

// 身份证号校验
/^[1-9]\d{5}(18|19|20)\d{2}((0[1-9])|(1[0-2]))(([0-2][1-9])|10|20|30|31)\d{3}[0-9Xx]$/

// URL验证
/^(((ht|f)tps?):\/\/)?[\w-]+(\.[\w-]+)+([\w.,@?^=%&:/~+#-]*[\w@?^=%&/~+#-])?$/

// 非负数
/^[+]?\d+(\.\d+)?$/

// emoji表情验证
/(\ud83c[\udf00-\udfff])|(\ud83d[\udc00-\ude4f\ude80-\udeff])|[\u2600-\u2B55]/g

// 密码校验，8-20位，字母、数字、特殊字符至少包含两种
/(?!^\d+$)(?!^[A-Za-z]+$)(?!^[^A-Za-z0-9]+$)(?!^.*[\u4E00-\u9FA5].*$)^\S{8,20}$/

// 密码校验，8-20位，必须包含大写字母、小写字母、数字、特殊符号(!@#$%&?*,.)
/^(?=.*[0-9])(?=.*[A-Z])(?=.*[a-z])(?=.*[!@#$%&?*,.])[0-9a-zA-Z!@#$%&?*,.]{8,20}$/
```
